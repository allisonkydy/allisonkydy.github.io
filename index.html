Digital Wasteland

React | react-vis | Django | PostgreSQL

Inspired by early text-based adventure games and multi-user dungeons, Digital Wasteland is a virtual environment with a hazardous apocalyptic setting. Players navigate through 100 procedurally generated rooms, interact with items, chat with other players, and steal items from other players' inventories. Real-time Pusher updates power communication and player actions. Built in 4 days with a remote 4-person team. I was responsible for building the Django server, writing the room generation algorithm, and augmenting game functionality on the React client.

MoodBloom

React | GraphQL | Apollo-Boost | Node.js/Express | Passport.js | MongoDB | Jest | Styled Components | Ant Design

A progressive web app for therapy patients suffering from mental health issues to track their mood and other behaviors, discover correlations, and complete daily tasks to improve their mood. Designed primarily for mobile devices. Developed over 8 weeks with a remote team of web developers and UX designers. I worked primarily on the front end building React components and using GraphQL queries with Apollo to dynamically display user data. I also set up tracking using Google Analytics and created custom styling following design prototypes.

Lambda Treasure Hunt - Team JAM

React | Python | Styled Components | Django | PostgreSQL

A unique project run by Lambda School to top off the computer science curriculum. Completing the treasure hunt required knowledge of graph traversal, data structures, computer architecture, and blockchain technology to create a world graph, find optimal paths, mine Lambda coins, gather treasure, and hunt the golden snitches. Our team created a React client to interact with differnt game features, a Django server to house the world map and pathfinding algorithm, and several looping algorithms in Python for different game modes. I built out the Django server, added game modes to the front end, and converted an 8-bit computer emulator to automatically parse machine code.
